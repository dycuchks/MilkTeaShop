# 遇见·好茶 MilkTeaShop 奶茶在线点单系统
✅ ASP.NET WebForm + C# + SQL Server 课程设计完整版项目
> 软件工程专业ASP.NET的期末大作业源码，开箱即用，新手可直接部署交作业。

## 📌项目介绍
B/S架构奶茶线上点餐系统，分为**前台用户端 + 后端管理员**双模块，满足课程设计全部评分需求，市面上同类课设常用模板。

### 🎯核心功能
|模块|功能详情|
|----|----|
|用户前台|用户注册登录、饮品分类浏览、商品搜索、加入购物车、修改购物车数量、订单结算下单|
|管理员后台|商品新增/编辑/下架、分类管理、全部订单查看+修改订单状态、用户信息管理|

## ⚙开发环境
- IDE：Visual Studio 2019/2022
- 框架：ASP.NET WebForms(.NET Framework)
- 数据库：SQL Server2012及以上
- 数据访问：原生ADO.NET
- 前端：HTML+CSS+JS

## 🚀一键部署步骤（3步运行）
1. 使用VS打开`MilkTeaShop.sln`解决方案
2. 打开SQL Server，新建数据库，执行项目内`TeaDB.sql`脚本自动建表+初始化测试数据
3. 修改Web.config内数据库连接字符串，F5启动项目
